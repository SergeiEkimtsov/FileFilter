# FileFilter

File filter application

Утилита для фильтрации текстовых файлов с поддержкой различных режимов работы и статистики.

Требования:

Java 17 или выше
Maven 3.5+
JUnit 5 и AssertJ для тестирования (включены в зависимости)

Возможности:

Обработка текстовых файлов — чтение, анализ, преобразование данных и запись результатов в файлы, вывод статистики в консоль

Автоматическое разделение данных на три категории:
Строки (только буквы и пробелы)
Целые числа
Числа с плавающей точкой (включая научную нотацию, например 1.5e-1)

Поддержка аргументов командной строки для настройки

Вывод статистики краткой или полной в консоль

Аргументы командной строки
-a              режим записи в файл (default: overwrite или append добавление)
-s              вывод в консоль краткой статистики
-f              вывод в консоль полной статистики
-o "dir"  	    указание директории для результатов
-p "pref"       указание префикса для выходных файлов

Выходные файлы:
Программа создает три файла в указанной (или текущей) директории при условии наличия обработанных данных:

<префикс>string.txt — строки
<префикс>integer.txt — целые числа
<префикс>float.txt — числа с плавающей точкой

Установка и сборка:

Клонирование репозитория
git clone https://github.com/SergeiEkimtsov/FileFilter.git
cd FileFilter

Сборка проекта
mvn clean package

После успешной сборки в папке target/ появится исполняемый файл util.jar.

Запуск:

Пример с одним файлом:
java -jar target/util.jar data.txt

Пример с параметрами:
java -jar target/util.jar -a -f -o output -p sample- data1.txt data2.txt

Пример простой фильтрации:
java -jar target/util.jar input.txt
Создаст файлы: string.txt, integer.txt, float.txt
